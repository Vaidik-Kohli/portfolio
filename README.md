# Vaidik — Portfolio

My personal portfolio website built with HTML, CSS, and vanilla JavaScript.

**Live:** [https://vaidik-kohli.github.io/Portfolio/](https://vaidik-kohli.github.io/Portfolio/)

## Tech Stack
- **HTML5** — Semantic structure
- **CSS3** — Custom properties, Grid, Flexbox, glassmorphism
- **JavaScript** — IntersectionObserver animations, smooth scroll, mobile menu

## Features
- Dark minimal design with blue/purple accent gradient
- Scroll-triggered fade-in animations
- Responsive: mobile, tablet, desktop
- Typing effect on hero
- Active section highlighting in navbar
- Glassmorphism navbar on scroll
- Contact form (Formspree-ready)
- Downloadable resume button

## Sections
1. Hero — Name, tagline, CTA, social links
2. About — Bio + highlight cards
3. Skills — Dev, Design, Other categories
4. Projects — 3 placeholder project cards (replace with your own)
5. Blog — 3 placeholder article cards
6. Contact — Info + form

## Setup
1. Clone this repo
2. Open `index.html` in your browser — no build tools needed

### GitHub Pages Deployment
1. Go to **Settings → Pages**
2. Source: **Deploy from a branch**
3. Branch: **main** / **(root)**
4. Save — your site will be live at the URL above

## Customization
- **Colors:** Edit CSS variables in `:root` at the top of `styles.css`
- **Content:** Edit text directly in `index.html`
- **Projects:** Duplicate a `.project-card` block and fill in your project details
- **Resume:** Replace `assets/resume.pdf` with your actual resume
- **Contact form:** Replace `your-form-id` in the form action with your [Formspree](https://formspree.io) endpoint
